#krep

CLI Tool that replicates the basic functionality of `grep` in Rust.

## Installation

```bash
cargo install https://github.com/karim-w/krep
```

## Usage

```bash
{process} | krep <pattern>

```

## Example

```bash
$ echo "Hello World" | krep Hello
Hello World
```
